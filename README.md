# Installation
`git clone https://github.com/Ahmed-Z/arp-scanner`<br>
`cd arp-scanner` <br><br>
After downloading you have to install dependencies:<br>
`pip3 install -r requirements.txt`

# How to use
`python3 arp-scanner.py --network 10.0.2.0/24`
